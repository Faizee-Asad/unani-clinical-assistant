# BUMS / Unani Curriculum Map for Skill Use

This is a practical map, not an official syllabus reproduction. Verify with the current NCISM curriculum and institutional syllabus.

## Foundational support

- Kulliyat/basic principles
- Tareekh-e-Tib/history
- Mizaj and Akhlat concepts
- Asbab-e-Sitta Zarooriya
- Anatomy/physiology correlation

Outputs: concept maps, short notes, viva questions, tables.

## Clinical support

- Moalajat/medicine
- Ilmul Jarahat/surgery
- Ilaj Bit Tadabeer/regimenal therapy
- Gynecology/obstetrics/pediatrics where relevant
- Emergency and referral awareness

Outputs: case presentations, OPD notes, red-flag checklists, referral letters.

## Pharmacy and drug support

- Ilmul Advia
- Ilmul Saidla wa Murakkabat
- Single-drug monographs
- Compound formulation documentation

Outputs: monographs, verification tables, safety fields.

## Internship/clinic support

- Case-taking
- Patient communication
- Follow-up plans
- Consent forms
- Referral summaries
