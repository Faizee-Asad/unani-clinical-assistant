# Evidence and reference guidance

## Never fabricate sources

If the user asks for references and none are provided, say references must be verified. Do not invent page numbers, book chapters, trials, or authors.

## Separate source types

1. Classical Unani source or concept
2. Prescribed textbook/institutional teaching
3. Pharmacopeia/formulary source
4. Modern biomedical evidence
5. Local clinical protocol
6. Practitioner judgment

## Language by evidence strength

- Strong evidence: “Evidence supports…” with citation.
- Limited evidence: “Limited evidence suggests…” with citation.
- Classical-only: “Classically described as…” and verify.
- Unknown: “Evidence/safety status not established here.”

## Source verification checklist

- [ ] Exact title/source identified
- [ ] Edition/year verified
- [ ] Page/chapter verified where applicable
- [ ] Drug identity verified
- [ ] Contraindications checked
- [ ] Patient-specific risks considered
- [ ] Modern evidence accurately summarized
- [ ] No fabricated citation details
