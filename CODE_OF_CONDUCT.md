# Code of Conduct

Be respectful. This project discusses traditional medicine, modern clinical safety, patient-facing workflows, education, and documentation. Critique claims and workflows, not people.

## Expected behavior

- Put patient safety first.
- Respect different professional backgrounds.
- Use careful, evidence-aware language.
- Avoid harassment, insults, personal attacks, or discriminatory comments.
- Do not pressure contributors to share private patient or institutional information.

## Project values

- Patient safety
- Privacy
- Humility
- Transparency about uncertainty
- Respect for Unani educational context
- Clear separation between documentation support and clinical decision-making
